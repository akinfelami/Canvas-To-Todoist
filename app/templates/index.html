Canvas-To-Todoist

Automatically sync your canvas assigments into Todoist

HOW TO USE

  • Generate a Canvas API key
    • Log into your institutions canvas portal
    • Navigate to Account -> Settings
    • Scroll down to approved Integrations and generate a new access token
    • Copy your access token
  • Generate a Todoist API Key
    • Log in your Todoist account
    • Go to Settings -> Integrations
    • Copy your API key located at the bottom

WARNING: Your API Keys are private and should be kept safely

We'll never store nor share your api keys with anyone else
